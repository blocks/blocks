# Components

Blocks come with a kit of components that are used in the editor and
can be used as standalone React components.

## Installation

```sh
yarn add @blocks/kit
```

## Usage

```js
import React from 'react'
import { BackgroundGradient } from '@blocks/kit'

export default () => (
  <BackgroundGradient>
    <h1>Hello, world!</h1>
  </BackgroundGradient>
)
```
